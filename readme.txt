OATools_2B is an Add-In for ArcMap version 10.2. and higher. It is a set of orientation analysis tools for tectonic data processing in the mapping environment of ArcGIS. The code is created in Microsoft Visual Basic 2010 Express (VB.NET). Here is the project in Microsoft Visual Studio. Compiled Add-In can be downloaded here: https://is.muni.cz/www/175417/OATools.html Detailed guide is included with installation notes.
The application source code is shared under the GPL-3 licence.
